# Decision Trace Examples

These are the concrete trace artifacts shown in the hackathon video.

They are copied from:

```text
artifacts/micro_world_demo/latest/
```

The trace shape is:

```text
score_info.json
candidate_scores.json
selected_action.json
outcome_after_execution.json
run_manifest.json
```

Use these files to verify the video claim that the robot-view demo writes a
replayable WorldForge-style evidence trail for every scored action.
